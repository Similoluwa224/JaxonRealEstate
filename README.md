**Jaxon Real Estate Sales & Operations Analysis**
**Project Overview**

This project analyzes a real estate sales and operations dataset using Microsoft SQL Server to perform data preparation, cleaning, and normalization before preparing the dataset for business intelligence analysis. 
The dataset contains information about customers, real estate agents, agencies, property listings, and property transactions. The goal of this project was to transform raw operational data into a clean, structured, and analysis-ready dataset that can support real estate business insights.
The project demonstrates a complete SQL data workflow, including:

**Database creation**
- Data importation
- Data cleaning and validation
- Missing value handling
- Statistical imputation
- Data normalization
- Dataset exportation for analytics

The final cleaned dataset was exported for further analysis and dashboard development using Microsoft Power BI.
